# Getting Started

## Purpose

이 문서는 `Codex`를 처음 사용하는 사람도 `sb-codex-tool`을 순서대로 따라
쓸 수 있게 해 주는 입문용 시작점입니다.

핵심 목표는 다음 두 가지입니다.

- 어떤 순서로 명령을 실행해야 하는지 바로 이해하기
- 실제 예시를 보고 첫 work cycle을 혼자 열고 닫아 보기
